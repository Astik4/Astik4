### 🛡️ Featured Projects

#### [🛡️ CyberShield SOC Portal](https://github.com/Astik4/Cyber-Defense-portal)
An end-to-end, enterprise-grade Security Operations Center (SOC) simulation platform featuring a modern command-and-control dashboard.
* **Key Features**: Real-time packet sniffing (Scapy), reactive intrusion detection system (IDS) rule compiler, active defense controls (lockdown mode, honeypot decoys, IP banning), live diagnostic tools (interactive visual traceroute, multi-port scanner, attack simulator), and persistent SQLite logging.
* **AI Integration**: Powered by Google Gemini to analyze alerts and provide immediate mitigation summaries.
* **Tech Stack**: `Python`, `Flask`, `React`, `Socket.io`, `SQLite`, `Scapy`, `HTML/CSS/JS`.

#### [🔍 VigilScan: Risk-Scoring Port Scanner](https://github.com/Astik4/VigilScan)
A portfolio-grade Python CLI tool for network port scanning, service detection, and vulnerability assessments.
* **Key Features**: Multithreaded TCP connect scanning, banner grabbing (SSH, FTP, SMTP, HTTP), real-time National Vulnerability Database (NVD API v2.0) CVE lookups, locally cached scans via SQLite to bypass API rate limits, and quantitative host-level risk scoring.
* **Tech Stack**: `Python`, `SQLite`, `NVD API`, `CLI`.

#### [🩸 LifeFlow: Blood Donation Management System](https://github.com/Astik4/Life-FLow)
A web-based blood donation finder designed to bridge the gap between donors and recipients.
* **Key Features**: Secure admin boundaries with session-based cookie authentication, donor-recipient matching based on location and blood group compatibility, cascade deletions, and a dynamic SPA-like web interface.
* **Tech Stack**: `Python`, `Flask`, `MySQL`, `HTML`, `Vanilla CSS`, `JavaScript (Fetch API)`.
